# Example Soundpacks

## `SampleClickPack`

Small synthetic soundpack that can be imported through the app Preferences window.

It demonstrates:
- The folder convention (`typing/`, `enter/`, `backspace/`, etc.)
- Multiple sounds in `typing/` (random pool behavior)
- One `.ogg` file (`typing/type_click_3.ogg`) to exercise Ogg Vorbis -> WAV import conversion

Import path:

- `/Users/mikeonator/Documents/Code/MacOSTypingSounds/Examples/SampleClickPack`

Contents:

- `typing/` (3 files: 2 WAV, 1 OGG)
- `enter/`, `backspace/`, `tab/`, `space/`, `escape/`
- `launch/`, `quit/`

All audio files are synthetic tones/clicks generated for this repo (no copyrighted game audio).
